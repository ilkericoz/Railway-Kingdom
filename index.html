Game Rules

  • Your goal is to create a continuous circular railway without crossing or branching tracks.
  • Bridges only allow straight tracks, while mountains force a 90-degree turn.
  • Railways cannot be placed on an oasis.
  • Left-click: cycle through track types (straight, corner, none).
  • Right-click: rotate track orientation (for straight and corner pieces).
  • The game ends when a valid, closed loop is created covering applicable tiles.
